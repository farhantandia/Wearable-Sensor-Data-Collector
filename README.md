# SensorDataCollectorwithFilter
 
Wearable sensor data collector with timer and preset sampling frequency and label.
Tested using wear OS Ticwatch Pro and Huawei 2.

Feature:
- Auto save 
- Water lock mode
- Saved as CSV File
- Adjustable sampling rate and action type
- Open to modify.
